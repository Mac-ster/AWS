1	What are security groups?
Ans.	•	Security Groups are fundamentals of network security in AWS.
•	They control how traffic is allowed into or out of our EC2 instances.
2	What are inbound rules?
Ans.	Inbound rules are used to control incoming network traffic the traffic coming from external sources ex: Internet, to specific device, server or network
3	What are outbound rules?
Ans.	Outbound rules are used to control the outgoing traffic from specific device, server or user from a network to external sources.
4	How to create security groups?
Ans.	Either while creating the resources or by going to Security and creating Security group and assigning with appropriate VPC. Need to add the required Inbound and Outbound Rules
