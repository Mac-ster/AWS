1	What is network ACL/ NACL?

Ans.	Yes
3	Is NACL supports for both allow and deny rules?
Ans.	Yes, it is a stateless service both allow and deny rule needs to applied
4	Is NACL stateless?
Ans.	Yes
5	Is NACL process rules in number order when deciding whether to allow traffic?
Ans.	Yes, it works in sequence order of rules define.
6	Which number gets the first priority?
Ans. 1
7	Is NACL apply on subnets and all instances under the same subnet?
Ans.	Yes, unlike security group which works on instance level, NACL works at subnet level and all the instances under that subnet.
8	Which default network ACL created  for VPC?
Ans.	100 & *
9	Is rule “100” and “*” are outbound?
Ans.	These are both outbound and inbound rules
10	What are the differences between security groups and NACL?
Ans.	 Both are fundamentals of network security in cloud, unlike Security groups which operates at instance level, NACL operates at Subnet level
